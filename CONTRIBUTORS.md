## How to Become a Contributor

We welcome contributions from everyone! Here's how you can contribute:

### Code Contributions

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request

See [CONTRIBUTING.md](CONTRIBUTING.md) for detailed guidelines.

### Non-Code Contributions

You can also contribute by:

- **Reporting Bugs**: Help us identify and fix issues
- **Suggesting Features**: Share your ideas for improvements
- **Improving Documentation**: Help make our docs better
- **Answering Questions**: Help other users in discussions
- **Testing**: Test new features and provide feedback
- **Spreading the Word**: Share the project with others
